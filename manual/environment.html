3. 必要な環境と準備

3.1 最低限必要なもの

  • モダンブラウザ（Google Chrome, Microsoft Edge, Firefox, Safari など）
  • テキストエディター（VS Code, Atom, メモ帳でも可）
  • ゲームファイル一式（GitHub からダウンロード、または git clone）

3.2 推奨ツール

  • Git：変更履歴を記録して、いつでも元に戻せます。
  • Live Server 拡張：編集した内容をブラウザで即確認できます。
  • 画像編集ソフト：アイテムや背景画像を作りたいときに便利です。
注意: インターネットに接続できない環境でも遊べますが、外部からダウンロードした素材を使うときは著作権やライセンスに気をつけてください。

3.3 ディレクトリの準備

  1. 開発用のフォルダーを決め、ゲーム一式を配置します。
  2. Git を使う場合は git init でリポジトリを作成し、初期状態をコミットします。
  3. バックアップ用にクラウドストレージや外部ドライブへのコピーも検討すると安心です。

3.4 ブラウザ設定

  • 開発者ツール（F12）をすぐに開けるようショートカットを覚えておきます。
  • キャッシュを無効化する拡張や設定を利用すると、変更がすぐに反映されます。
  • レスポンシブデザインを確認するため、デバイスモードの使い方を覚えておきましょう。
