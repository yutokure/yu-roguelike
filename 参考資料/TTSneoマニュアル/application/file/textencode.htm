TTSneoで対応している文字コード

TTSneoでは、Shift-JIS/EUC/JIS/UTF8の各文字コードのファイルを開いたり保存したりすることができます。

通常の操作では、Shift-JIS形式で保存されます。

　

指定した文字コードでテキストファイルを開く

下のように指定すると文字コードがEUC/JIS/UTF8の各文字コードのファイルを開くことができます。

内容は、開け（「文章.txt」、EUC）
内容は、開け（「文章.txt」、JIS）
内容は、開け（「文章.txt」、UTF8）

使用されている文字コードを自動的に認識して開く場合は、次のように書きます。

内容は、開け（「文章.txt」、自動）

　

指定した文字コードで保存する

下のように指定すると文字コードがEUC/JIS/UTF8形式のファイルとして保存することができます。

「文章.txt」へ「こんにちは」をEUCで保存
「文章.txt」へ「こんにちは」をJISで保存
「文章.txt」へ「こんにちは」をUTF8で保存

文字コードを指定しないと、Shift-JIS形式で保存します。
