TTSneoについて

ソフト名 TTSneo (Technology Terminal Script neo)
種別 フリーソフト (無料で利用できます)
著作 ゆうと
動作環境 Microsoft Windows 7RC/Vista/XP/2000/Me/98 (95/NT 4.0)
の各エディション　のいずれかが動作するパソコン

※Visual Basic 6.0ランタイムをインストールしてください。
　提供されているすべての機能を使うには、IE5.5以上が必要です
　（IE5.5がインストールされてなくても動作します）
※Windows 95/NTでは一部の機能が使えない場合があります
サポート 掲示板とメール
　TTSneo以外の質問（VBの使い方やコンパイラの作り方についてなど）
　にはお答えできませんので、ご了承ください。
ウェブサイト http://tts.utopiat.net/
メールアドレス 　（重要な連絡のみ）
メッセンジャは行っておりません。質問は掲示板でのみ受け付けております
開発開始 1999年11月28日

　

動作環境

TTSneoの動作環境は、次の通りです。

推奨動作環境

Windows98以降が動作するパソコン
Internet Explorer 6.0SP1以上
TTSneo公式サイトで公開しているVisual Basic 6.0ランタイム

最低限の動作環境

Windows 7/Vista/XP/2000/NT4.0/Me/98 (95/NT4.0)　のいずれかが動作するパソコン
Visual Basic 6.0ランタイム

作成したプログラム／スクリーンセーバーが動作する環境

Windows 7/Vista/XP/2000/NT4.0/Me/98 (95/NT4.0)
TTSneo公式サイトで公開しているVisual Basic 6.0ランタイム

動作テスト環境

Windows 7 Professional
Visual Basic 6.0ランタイム(SP6)
Internet Explorer 9.0

WindowsXP Professional SP3
Visual Basic 6.0ランタイム(SP6)
Internet Explorer 8.0

Windows98 Second Edition
Visual Basic 6.0ランタイム(SP6)
Internet Explorer 6.0 SP1

※Windows2000上で文字化けする場合は、TTSneo公式サイトで公開しているVisual Basic 6.0ランタイムをインストールすることで解消します。
※日本語版Windowsで動作確認しています。外国版では動作しません。
