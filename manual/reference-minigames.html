20. MiniExp モジュール一覧

P.1 マニフェストの仕組み

MiniExp のモジュール情報は games/manifest.json.js を通じて提供されます。main.js は loadMiniManifestOnce() 内で次の優先順位に従ってマニフェストを読み込みます。

  1. window.MINIEXP_MANIFEST が既に定義されていればそれを使用。
  2. games/manifest.json.js を <script> として読み込み（file:// 環境対応）。
  3. HTTP サーバー環境では games/manifest.json を fetch で取得。
  4. 上記がすべて失敗した場合は、ソースコードに内蔵されたフォールバック配列を利用。

詳細な実装は main.js 6439–6491 行を参照してください。各モジュールは ID、名称、エントリファイル、バージョン、 カテゴリ、EXP の付与ルールなどを定義します。

P.2 カテゴリ別モジュール一覧

収録済み MiniExp MOD をカテゴリごとに整理しています。ID は registerMiniGame() に登録する値と一致し、 エントリ列のスクリプトを読み込むとモジュール本体が提供されます。
