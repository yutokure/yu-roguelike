文句一つ言わず・・・

パソコンは、同じことを何度も何度も、文句一つ言わずやってくれる道具です。

こうやって、マニュアルを読んでいる間にも、「マウスは動いた？・ボタンを押した？キーボードを押した？・・・・」ということを永遠に何万回も、繰り返してくれています。

ここでは、パソコンに「繰り返し」処理をやらせる方法を紹介します。
この命令を使って、パソコンをこき使ってくださいね！

　

同じ命令を何度も繰り返したい

ねお　「今回は、同じプログラムを何度も実行する方法を紹介するね。」

とも　「分かった。どうやれば、何度も実行できるの？」

ねお　「「繰り返せ」命令を使います。次のプログラムを見て下さい。」

３回「「ワン」を表示」を繰り返せ

とも　「「ワン」と３回表示されたね」

ねお　「このように

○○回「〜〜」を繰り返せ

と命令すると、命令が指定した数字だけ繰り返されます。」

とも　「○○に回数を入れて、〜〜に命令を入れるんだね」

ねお　「そのとおり。○○回の部分を省略すると、〜〜の命令をずっと繰り返します。」

ねお　「〜〜にたくさんの命令を入れたい時は、下のように複数の行を使って書くこともできます。最後に「繰り返し終わり」を入れるのを忘れないでね」

３回繰り返せ
　　「ワン」を表示
　　待て　１
繰り返し終わり

とも　「繰り返せ・・・繰り返し終わり　だね」

※「待て　[秒数]」命令は、指定した秒数だけプログラムを一時停止する命令です。

　

繰り返しから抜ける

とも　「繰り返ししている途中で、抜け出すことはできないの？」

ねお　「いい質問だね！！　「繰り返しから抜ける」と命令すると、指定した回数でなくても、繰り返しを止めることができます。」

とも　「なるほど！」

例　一度しか「ワン」と表示されません。

繰り返せ
　　「ワン」を表示
　　繰り返しから抜ける
繰り返し終わり

　

「繰り返せ」命令を使った絵を描くプログラム   

ねお　「「繰り返せ」命令で絵を描いてみましょう。」

とも　「えっ　絵が描けるの？」

ねお　「もちろんだよ。TTSneoは簡単に絵を描く方法があるんだよ
詳しくは、この後説明するから、しっかり聞いていてね」

ウィンドウ１を表示
クリア
出てこい
位置は　中央
ペンを下ろせ
６回を繰り返せ
　　右へ　１２０
　　前へ　３０
　　左へ　６０
　　前へ　３０
繰り返し終わり

とも　「よし、実行っと。　あっ！カメさんが絵を描いている！」

ねお　「このプログラムでは、カメを使って、「６回を繰り返せ」から「繰り返し終わり」にある命令を６回実行しています。」

このプログラムに出てくる命令については、機能編のカメを使って絵を描くで紹介します。
とりあえずは、無視して次へ進んでください。

　

数字をカウントしながら繰り返す

ねお　「「繰り返せ」命令では、指定した変数に繰り返した回数をカウントすることもできます。」

とも　「じゃぁ　わざわざ変数を使って足し算をさせなくていいんだね？？　(ヨクワカラナイ)」

ねお　「そうだね。この後紹介する配列を使うときや、数字によって変化する処理を作りたい時などに便利です。」

○○回、〜〜を□□に△△からカウントして繰り返せ

○○と〜〜は、さっきと同じです。

□□

カウントする変数を指定します。
繰り返されるたびに、この変数の数字が１ずつ増えます。

△△

変数のはじめの数を指定します。
その数からカウントします。

ねお　「複数行で使う場合は、繰り返せの最初の行に「〜に〜からカウントして」を入れてください。」

ウィンドウ１を表示
１０回、数字に１からカウントして繰り返せ
　　数字を書け
　　待て　０．５
繰り返し終わり

とも　「ふーん。まあ、配列の時になったら、勉強するよ！」

　

練習問題っ！

練習問題をやってみましょう。
これができれば、まずは簡単なプログラムが作れるようになります。

問題１

「入力」関数を使って、数字を入力すると、その数になるまで数字をカウントするプログラムを作ってみましょう。

ヒント
　例えば、「3」と入力すると「1回」、「2回」、「3回」と表示されるようにしましょう。
　「入力」関数の使い方は、前回の４．で紹介しています。

問題２

次のプログラムは、パスワードを確認するプログラムです。
正しいパスワードを入力すると、「成功」と表示され、３回間違えると終了します。

(　　　　)に当てはまるプログラムを考えて見ましょう。

答えは、「password」
(　　　　)
　　結果は、入力（「パスワードを入力してください」）
　　もし　結果は、答え　なら
　　　　「成功」を表示
　　　　繰り返しから(　　　　)
　　もし終わり
(　　　　)

※「もし」〜「もし終わり」の部分は、結果がpasswordだったときに動くプログラムです。
次の「６．判断する　『もし』」で詳しく説明します。
