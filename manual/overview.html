2. ゲーム全体の概要

2.1 Yuローグライクとは

Yuローグライクは、ダンジョンを選んで冒険するブラウザゲームです。HTML・CSS・JavaScript だけで動くため、 Web ブラウザがあればどこでも遊べます。ゲーム内ではダンジョンの種類や難易度を選び、登場する敵やアイテムを楽しめます。

2.2 特徴

  • シンプルな構造で、カスタマイズしやすい。
  • ミニゲームや開発者向けツールが豊富。
  • レベル100から開放される SP とスキルシステムで高難度の攻略を後押し。
  • データファイルが分かれており、ゲームバランスを調整しやすい。
  • 初心者でも理解できるよう、コメントや設計書が多数用意されている。

2.3 想定するユーザー

  • ゲームを遊びたいプレイヤー
  • ダンジョンや敵を追加したい制作者
  • JavaScript を学びながらゲームを作りたい初心者
  • 教育目的で教材として使いたい人

2.4 進行の流れ

  1. タイトル画面で難易度・世界・ダンジョンを選択する。
  2. 詳細カードで推奨レベルや説明を確認する。
  3. ダンジョンへ入って探索・戦闘・報酬獲得を楽しむ。
  4. 必要に応じてミニゲームやツールで準備や調整を行う。

以上の流れはデフォルト構成であり、データファイルを書き換えることで自由にアレンジすることが可能です。

2.5 カスタマイズ可能な要素

  • ダンジョン： 種類・説明文・推奨レベル・報酬テーブルなどを JSON ファイルで変更できます。
  • 難易度設定： ダメージ倍率や敵配置などを調整して独自のモードを作れます。
  • ビジュアル： CSS を書き換えて配色やレイアウトを自由に変更できます。
  • 拡張機能： JavaScript を追加して新しい UI コンポーネントやタブを実装できます。
