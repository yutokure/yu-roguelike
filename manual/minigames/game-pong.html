ミニゲーム: ピンポン

概要

左側のパドルを操作してボールを跳ね返し、右側の CPU パドルとのラリーで先に 5 点を獲得することを目指すクラシックなポン風ゲームです。 自分がポイントを取るたびに経験値が加算され、マッチに勝利すると難易度に応じた追加経験値を獲得できます。

起動と準備

  1. ミニゲームハブで ピンポン を選ぶと、難易度選択ダイアログが表示されます。
  2. EASY／NORMAL／HARD のいずれかを選択して開始すると、中央にボール、左右にパドルが配置された状態でゲームが始まります。
  3. 画面上部には現在のスコア、下部には「First to 5」と選択中の難易度が表示されます。

難易度はホストメニューのリスタート操作からいつでも変更可能です。

操作方法

  • ↑ / ↓：パドルを上・下に移動します。
  • W / S：キーボードの代替入力としても利用できます。

押し続けると連続的に移動し、パドル位置は自動的に画面端で制限されます。

ルールと得点

  • ボールを相手側のゴールラインまで通過させると 1 点を獲得し、難易度に応じて +1 / +3 / +10 XP が付与されます（EASY → HARD）。
  • 先に 5 点に到達した側がマッチの勝者となり、勝利時に追加で +20 / +100 / +400 XP を獲得します。
  • CPU に得点されると相手のスコアが増えますが、プレイヤー側の得点と経験値はリセットされません。
  • マッチが決着するとスコアが 0-0 にリセットされ、同じ難易度で次の試合が自動的に始まります。

難易度の違い

難易度はパドルの高さ・移動速度、ボール速度、CPU の追従精度と報酬量に影響します。

難易度 パドル高さ パドル移動速度 ボール初速 CPU 追従係数 ポイント XP 勝利 XP
EASY 70 px 7 px / フレーム 4.0 px / フレーム 0.25 +1 +20
NORMAL 60 px 8.5 px / フレーム 4.8 px / フレーム 0.18 +3 +100
HARD 52 px 10 px / フレーム 5.6 px / フレーム 0.10 +10 +400

ゲームの流れ

ボールはラリーのたびにわずかに加速し、パドルの中央付近に当てると真っすぐ返球、端に近いほど大きな角度で返球されます。 壁に当たると反射し、CPU パドルはボールの位置を追跡しながら遅延して移動します。CPU の追従係数が小さいほど反応が鋭くなります。

攻略のヒント

  • ボールがパドルの端に当たるよう角度を付けると、CPU が追いつきにくいコースを作れます。
  • ラリーが長引くとボール速度が上がるため、カーソルを小刻みに動かして待ち構えると反応しやすくなります。
  • HARD では CPU の追従が速いので、パドル位置を早めに調整し、返球角度で勝負しましょう。
