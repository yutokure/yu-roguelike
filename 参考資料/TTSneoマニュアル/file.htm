インストールについて

TTSneoマニュアル

インストールする

TTSneoは、必要なファイルを一つのファイルにまとめて公開しています。
ファイルを開くと、インストールが始まります。
メッセージにしたがってインストールを行ってください。

※Windows 2000以降では、管理者権限(Administrator)でログインして行うことをおすすめします

また、TTSneoを使うには、Visual Basic 6.0ランタイムが必要です。
初回起動時に表示される画面からインストールするか、
Visual Basic 6.0ランタイムをダウンロードしてください。

TTSneoデザイナが起動できない場合は、FAQの「起動しないときは」をご覧ください。

　

フォルダ構成

インストールすると下のようなフォルダが作成されます。

「サンプル」フォルダ

TTSneoで作ったサンプル集です。
プログラムを作るにあたってのきっかけになりますので、絶対に見てください。

「マニュアル」フォルダ

今ご覧になっているマニュアルです。
Wordで開くことができる命令・関数一覧表があります。印刷してご覧ください。

「ツール」フォルダ

TTSneoデザイナから呼び出すツール（マクロ）があるフォルダです。
ここにTTSneoのプログラムをコピーすると実行できます。

「Template」フォルダ

TTSneoデザイナが使用するデータです。
新規作成のヘッダやテンプレートなどが保存されています。

「必要なDLL」フォルダ

TTSneoの一部の機能を利用するのに必要なDLL（VBMP3.dll、TransG32.dll、nMail.dllなど）の著作権情報と注意書きです。
作ったソフトを配布する際などには必ず読んで下さい。

「runtime」フォルダ

TTSneoをインストールしていないパソコンでもTTSneoで作ったプログラムを実行できるプログラム（ランタイム）です。
ランタイムの種類は、TTSneoデザイナから複数選択できます。

　

システムフォルダにコピーされるファイル

インストールすると、WindowsのシステムフォルダへTTSneoが使用する次のファイルをコピーします。

ファイル名 用途 形態
TTScript.dll TTSneoシステムDLL TTSneo専用
TTSExtra.dll TTSneo機能拡張 TTSneo専用
TransG32.dll PNG・JPEG関連 汎用ライブラリ
Vbmp3.dll MP3再生 汎用ライブラリ
nMail.dll メール送受信 汎用ライブラリ
TTSDirectX.dll DirectXプラグイン TTSneo専用
TTSSpeech.dll 音声合成プラグイン TTSneo専用

「汎用ライブラリ」とは、フリーソフトとして提供されているライブラリで、他のソフトウェアで利用されている場合があります。

　

アンインストールについて

TTSneoは、「コントロールパネル」の「アプリケーションの追加と削除」から削除できます。

１．レジストリ項目を削除する（アンインストール前に行ってください）

設定情報などのレジストリに登録されているTTSneoに関連する情報をすべて削除したい場合は、インストールしたフォルダにある「Setup.exe」を起動してください。

「設定の削除」を選ぶと、設定がすべて削除されます。
その後、TTSneoをアンインストールしてください。

２．TTSneo本体のアンインストール

「コントロールパネル」の「アプリケーションの追加と削除」を開き、「TTSneo」を選択して削除します。

※システムフォルダにあるTTSneoのみに関連するDLLは、削除されます
　TTSneoのフォルダが完全に削除されないことがありますが、必要なファイルがないことを確認して、個別に削除してください。
