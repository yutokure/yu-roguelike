ミニゲーム: ボンバーマン風

概要

固定マップ内で爆弾を設置し、破壊可能なブロックを吹き飛ばしながら敵の殲滅を目指すアクションミニゲームです。 爆風に巻き込まれないよう立ち回りつつ、通路を切り開いて敵を追い詰めるクラシックなボンバーマン風体験を再現しています。

準備とゲーム開始

  1. タイトルメニューから ボンバーマン風 を選択します。
  2. ホスト設定で難易度（EASY / NORMAL / HARD）を切り替えると、ブロック密度・持ち爆弾数・敵の数と移動速度などが変化します。
  3. 開始するとプレイヤーは左上の安全地帯からスタートし、内部にランダムに配置されたソフトブロックが表示されます。

進行中はホストの R キーショートカットが無効化されます。ゲームが終了すると再び使用できます。

操作方法

  • ← / → / ↑ / ↓：プレイヤーを移動します。斜め移動は自動的に速度が補正されます。
  • Space：現在位置に爆弾を設置します（設置直後は短時間すり抜け可能）。
  • ゲーム終了後の R：盤面をリセットし、難易度設定を維持したまま再スタートします。

マウス操作は不要で、キーボードのみで完結します。

ルールと得点

  • 爆弾は約 2.3 秒 の導火線が燃え尽きると縦横に爆風を伸ばします。ブロックに当たるとそこで停止します。
  • ソフトブロックを破壊すると +3 XP（スコアにも加算）、敵を爆風に巻き込むと +12 XP を獲得します。
  • 画面左上の HUD には残機、総スコア、生存中の敵数が常時表示されます。
  • 全ての敵を倒すと追加で +75 XP を獲得し、ステージクリアになります。
  • 爆風は他の爆弾を誘爆させます。連鎖を活用すると一気に通路を開きやすくなりますが、同時に自分が巻き込まれないよう注意が必要です。

ライフとゲーム終了条件

爆風や敵との接触で残機を 1 つ失います。残機が尽きるとゲームオーバーです。 残機が残っている場合はプレイヤー位置と敵の初期配置がリセットされ、破壊済みのブロックはそのまま維持されます。 全ての敵を倒すか、残機が 0 になった時点でプレイが終了し、画面中央に結果オーバーレイが表示されます。

難易度による主な違い

  • EASY：ソフトブロック密度が低く、初期爆弾数 2、爆風射程 3。敵は 3 体で移動速度も遅め、残機は 3。
  • NORMAL：標準的な密度と敵 4 体。爆弾数 2、射程 2、残機 3。
  • HARD：ブロック密度が高く敵は 5 体。プレイヤー速度が速い代わりに爆弾所持数 1・残機 2 と制約が厳しくなります。

難易度を切り替えても基本ルールは共通です。自分のプレイスキルに合わせて選択してください。

攻略のヒント

  • 爆弾設置直後は短時間だけ爆弾をすり抜けられます。安全地帯に戻る際に活用すると事故を防げます。
  • 敵との距離を確保しながら通路を整備すると、誘導して爆風に巻き込みやすくなります。
  • 角や壁際に敵を追い込むと逃げ道が限定されます。爆風の射程を見極めて、連鎖爆破で一掃しましょう。
