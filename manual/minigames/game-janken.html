ミニゲーム: じゃんけん

概要

CPU と 1 対 1 で勝敗を競うクラシックなじゃんけんゲームです。勝てば 10EXP を獲得し、 連勝を重ねると追加ボーナスが発生します。MiniExp の休憩枠として手軽に遊べるよう、 3 つの手を選ぶだけのシンプルな構成ながら、履歴や統計で結果を振り返れるようになっています。

準備とゲーム開始

  1. タイトルメニューのミニゲーム一覧から じゃんけん を選択します。
  2. カード UI が読み込まれたら、グー／チョキ／パーのいずれかのボタンを押してラウンドを開始します。
  3. 掛け声（「最初はグー…」「じゃんけん…」「ぽん！」）の後に CPU の手が決まり、即座に勝敗が判定されます。

ラウンドは時間制限なく繰り返せます。メニューへ戻るときは上部 UI の共通操作を利用してください。

操作方法

  • じゃんけんボタン（グー／チョキ／パー）：勝負に出す手を選びます。クリック（タップ）すると掛け声アニメーションが始まり、結果が表示されます。

キーボードショートカットは実装されていないため、マウスやタッチ操作でボタンを押してください。

ルールと得点

  • 勝利すると 10EXP を獲得します。連勝数に応じて以下のボーナスが加算されます：3 連勝で +4EXP、5 連勝で +8EXP、7 連勝以上で +15EXP。
  • 負けると連勝はリセットされ、ボーナスは入りません。あいこの場合は連勝を維持したまま次の手を入力します。
  • 結果カードには直近 6 ラウンドまでの履歴が表示され、最新の結果はカード上部にも強調表示されます。
  • 統計カードには勝ち／負け／あいこの累計、現在と最高連勝数、勝率がリアルタイムで更新されます。
  • CPU は完全ランダムではなく、プレイヤーの直前の手や連勝状況に応じて手を変化させる場合があります。長連勝時ほどカウンターを狙われやすくなる点に注意してください。

ゲーム終了条件

明確な終了条件はなく、プレイヤーが停止するまでラウンドが続きます。獲得した EXP は勝利時点で即座に加算されるため、 任意のタイミングで離脱しても結果が失われることはありません。

攻略のヒント

  • 連勝ボーナスを狙う場合は、直前ラウンドの自分の手を CPU が模倣してくるパターンや、カウンターを差し込んでくるタイミングを意識すると勝率を上げやすくなります。
  • あいこ時はストリークが継続するため、焦らず同じリズムで入力を続け、CPU の手癖が変化するかを履歴で確認しましょう。
  • 負けた直後は連勝補正がリセットされるので、次に取りやすい手を落ち着いて選び直し、再び連勝を積み上げるサイクルを意識してください。
