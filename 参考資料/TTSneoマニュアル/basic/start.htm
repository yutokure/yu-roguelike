TTSneoを使う

ねお　「よし！これからTTSneoを使ってプログラミングに挑戦しよう！」

とも　「オー！　　でも、どうやってプログラムを作るの？」

ねお　「TTSneoでプログラムを作るには、「TTSneoデザイナ」というソフトを使います。
まずは、それを出してみましょう。」

とも　「うん！　「スタート」メニューから起動するんだね。」

　

TTSneoデザイナを使う

「スタート」メニューから「プログラム(すべてのプログラム)」→「TTSneo」→「TTSneoデザイナ」をクリックします。

注意
初回起動時は、「Visual Basic 6.0ランタイム」をインストールする確認メッセージが表示される場合があります。
[はい]をクリックすると、TTSneoの公式サイトからVisual Basic 6.0ランタイムを自動的にダウンロードして、インストールを開始します。

制限ユーザで利用する場合など、同じメッセージが何度も表示され、TTSneoデザイナが起動できない場合は、FAQの「起動しないときは」をご参照ください。

　

「こんにちは」を表示させる

ねお　「TTSneoデザイナの画面が表示されると、文字を入力する部分が表示されます。ここにプログラムを入力していきます。
まずは、下のように入力してみてごらん。」

「こんにちは」を表示

とも　「これが、これからお世話になるTTSneoデザイナかぁ〜。
うん。入力したよ。」

ねお　「入力したら、実行してみよう。」

実行する

次に、プログラムを実行させます。

ツールバーにある[実行]ボタンをクリックするか、「実行」メニューから「実行」を選びます。


[実行]ボタン

　

とも　「おおお！　「こんにちは」って表示された！！」

ねお　「おめでとう〜。これで、とも君もプログラマへの道の一歩を踏んだね。
実は、F5キーを押すと、一発で実行できます。」

とも　「本当だ。こっちの方が楽かも。」

　

ショートカットキーを使おう

ファイルの保存やプログラムの実行は、メニューやツールバーをクリックして操作することができますが、ショートカットキーを利用することでキーを押すだけですばやく操作できます。

ファイルの保存 [Ctrl]+[S]
プログラムの実行 [F5]
検索 [Ctrl]+[F]
置換 [Ctrl]+[H]

ほかにもたくさんショートカットキーを用意しています。
ショートカットキーは、メニューの項目の右側に書かれていますので、いつも使う操作は、キーボードでできるようにしましょう。
