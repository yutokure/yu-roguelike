必要なソフトウェアのインストール

音声合成を利用する場合 (SAPI4.0を使用)

http://activex.microsoft.com/activex/controls/sapi/spchapi.exe
をダウンロードしてインストールします。(824KB)

http://www.microsoft.com/products/msagent/downloads/user.asp#tts
にアクセスして[Select a Text-to-speech engine] で [・・・　Japanese(3MB)] を選び
ダウンロードしてインストールしてください。

音声認識を利用する場合

Microsoft Speech SDK 5.1(68 MB)をダウンロードします。

Language Pack(Japanese&Chinese)(81.5MB)　をダウンロードします。（音声認識に必要）
インストール時に「Custom」を選び「Chinese」のチェックをはずしてください。
（中国版は不要なので）

以上の２つをインストールすると利用可能になります。
SAPI 5.1については、参考情報をご覧ください。

　

MS Agent(エージェント)機能を利用する場合

Microsoft OfficeやWindows Me/2000/XPがインストールされている場合、すぐに利用することができます。

それ以外のWindowsの場合は、下から必要なソフトウェアをダウンロードします。

http://www.microsoft.com/msagent/downloads/user.asp

さらに音声合成を利用する場合は、「音声合成を利用する場合」にあるファイルをダウンロードしてください。
