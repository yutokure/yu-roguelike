Feature Highlights

ブラウザですぐに冒険開始

HTML / CSS / JavaScript だけで完結。インストール不要で PC でもモバイルでも遊べます。

選べる世界と難易度

タイトル画面で難易度とダンジョンを選び、推奨レベルを確かめてから潜入。毎回変わる自動生成のマップに挑戦しましょう。

冒険者を育てるスキルツリー

レベル100から開放されるSPとスキルシステムで、高難度ダンジョン攻略に備えたビルドが組めます。

Gameplay

冒険の流れ

遊び方のマニュアルから、ステップごとの魅力をピックアップしました。

  1. 1. ダンジョンと難易度を選択

    推奨レベルや報酬を確認しながら、挑むダンジョンを決めます。

  2. 2. 自動生成の階層を探索

    ターン制で1マスずつ進み、敵との距離を測りながら戦略的に行動。

  3. 3. 宝箱イベントでリワード獲得

    黄金の宝箱ではタイミングゲームに挑戦し、レアな強化アイテムを狙えます。

  4. 4. 成果を持ち帰り再挑戦

    自動保存されたステータスをもとに、次の冒険へすぐアクセス。

For Creators

作る・学ぶ人へ

Yuローグライクはプレイヤーだけでなく、制作者や教育用途でも活用しやすい構造です。

カスタマイズ派の制作者に

JSONファイルでダンジョンや難易度を調整、CSSでテーマを刷新、JavaScriptでタブやUIを追加できます。

  • games / dungeontypes データの即時編集
  • UIレイアウトとスタイリングの自由度
  • 拡張用 API の豊富なリファレンス

学習教材として

コメントや設計書が揃っているので、JavaScriptの学習や授業の題材にも最適。

  • 構造を理解しやすいフォルダ構成
  • 手順を追えるマニュアルとチュートリアル
  • 演習しやすいミニゲームのサンプル

コミュニティでの共有

ツールズタブで開発を効率化し、ミニゲームの結果を本編に連携させるヒントも提供。

  • ブロックデータ編集などのユーティリティ
  • 共通データ管理のベストプラクティス
  • UI統一に役立つスタイルガイド
