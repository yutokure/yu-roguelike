ミニゲーム: ギャンブルホール

概要

ギャンブルホール は MiniExp の EXP をチップとして利用し、 ルーレット・パチンコスロット・ラッキーダイス の 3 種のチャンスゲームを 1 つの画面で切り替えて遊べる複合ミニゲーム MOD です。 ベットごとの純増減はセッション成績として記録され、直近 5 件の結果がログ化されるため、 連続プレイでも収支を把握しながら挑戦できます。

画面構成

  • サイドバー — 利用可能 EXP、セッション収支、最大獲得 EXP をまとめたサマリーカードと、 3 つのゲームを切り替えるナビゲーションボタンで構成されています。 ナビボタンはアイコンとサブテキスト付きで、現在選択中のゲームがハイライト表示されます。
  • ヘッダー — 中央部に現在の残高とベット入力欄、+10/+50/MAX の クイック調整ボタン、右側にセッション純利益と「最大獲得」表示を備えています。 セッション純利益はプラスで青緑、マイナスで赤に色分けされます。
  • メインパネル — 選択したゲーム専用の UI が表示されます。パネル上部にはゲーム名と概要が表示され、 内容に応じた操作ボタンが配置されます。
  • ステータス & 履歴 — 画面下部に現在の状況メッセージ、右側に直近 5 件までのプレイ履歴が表示されます。 履歴行は勝ちで緑、負けで赤に縁取りされ、ベット額と結果内訳が確認できます。

ベットと共通ルール

  1. サイドバーのナビボタンで遊びたいゲームを選びます。切り替えるとパネルタイトルと内容が更新されます。
  2. ヘッダーのベット欄に賭けたい EXP を入力します。入力値は自動で 0 以上、所持 EXP 以下にクランプされます。
  3. +10/+50 ボタンで素早く増額でき、MAX ボタンは現在の残高をそのままベット額に設定します。
  4. 各ゲームの開始ボタン（例: スピン / スタート / ロール）を押すと、 ベット額が即座に残高から差し引かれ、演出が始まります。
  5. 結果に応じた配当があれば自動的に EXP が払い戻され、ステータスに純増減と詳細が表示されます。

残高が不足している場合や演出中に再操作した場合は、警告メッセージが表示され実行されません。

ルーレット（欧州式 37 マス）

円形ホイールとボールのアニメーションで結果を表示するヨーロピアンスタイルのルーレットです。 ベットタイプは単一指定で、0 を含む 37 マス全体からランダムに結果が決まります。

ベットタイプ 的中条件 配当倍率 備考
赤 / 黒 対応する色に着地 x2 0（緑）はハズレ扱いになります。
緑 (0) 0 に着地 x36 単一ポケット指定と同倍率です。
偶数 / 奇数 0 以外の偶数 / 奇数に着地 x2 0 はどちらにもカウントされません。
番号指定 指定した番号に着地 x36 0〜36 から選択。選択時に番号リストが有効化されます。
  • スピン中はボタンが無効化され、ホイール停止まで次のベットは行えません。
  • 結果は中央表示に番号と色が示され、当たった番号のホイールマーカーがハイライトされます。

パチンコスロット

3 リール構成のシンプルなスロットです。7 / BAR / W / ☆ / 鈴 / 🍒 の 6 種のシンボルがランダムに停止し、 コンビネーションに応じて配当が決まります。スピン演出は約 1.2 秒で停止し、その間は連打できません。

役 条件 配当倍率
777 ジャックポット 7 が 3 つ揃う x18
BAR トリプル BAR が 3 つ揃う x12
その他トリプル 同一シンボルが 3 つ揃う x9
スター付きペア 同じシンボル 2 つ＋残り 1 つが ☆ x4
通常ペア 同じシンボル 2 つ（☆ 以外） x2
ハズレ 上記以外 配当なし

配当を得た場合は自動的に EXP が払い戻され、履歴にシンボル並びと役名が記録されます。

ラッキーダイス

3 個のサイコロを同時に振り、選択したモードの条件を満たせば配当を獲得します。ロールボタンを押すとサイコロが アニメーションし、最終結果（目・合計値）がステータス欄に表示されます。

モード 条件 配当倍率 補足
ハイ (11-18) 合計 11 以上かつゾロ目でない x2 ゾロ目（トリプル）は対象外です。
ロー (3-10) 合計 10 以下かつゾロ目でない x2 こちらもゾロ目は除外されます。
ラッキー7 合計がちょうど 7 x9 ゾロ目条件はありません。
バラバラ 3 つすべて異なる目 x4 ゾロ目を避ければ成立します。
ゾロ目 3 つすべて同じ目 x24 最高倍率。ハイ／ローとは独立判定です。

結果はサイコロの目アイコンで表示され、合計値と共に履歴へ記録されます。

ステータスと履歴の活用

  • ステータスパネルは勝利時に緑、敗北時に赤、警告時に黄で縁取られ、最新メッセージが常に再表示されます。
  • 履歴にはゲーム種別、ベット額、詳細（例: ルーレットの着目番号やスロットの絵柄列）が保存され、最大 5 件まで保持されます。
  • ゲームを開始するとセッション純利益と最大獲得 EXP がリセットされ、以降の結果が追跡されます。
