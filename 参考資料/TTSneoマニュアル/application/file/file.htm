ファイルのコピーと移動

ねお　「TTSneoでは、ファイルのコピー、移動、削除などの操作をすることができます。」

とも　「つまり、エクスプローラやマイコンピュータでマウスを使って作業していたことを、TTSneoでプログラムすることができるんだね！
それは便利そう。」

ねお　「まずは、ファイルの操作をする基本的な命令を紹介するね」

[もとのファイル名]を[コピー先のファイル名]へコピー
[もとのファイル名]を[コピー先のファイル名]へ移動
[ファイル名]を削除
[変更前のファイル名]を[変更後のファイル名]へ名前変更

とも　「アニメーションや確認メッセージを表示させないようにすることはできないの？」

ねお　「アニメーションを表示させたくないときや確認せずに上書きしたい時は、「すぐ」をつけて下さい。」

〜すぐコピー
〜すぐ移動
〜すぐ削除

　

関連付けられたファイルを開く

TTSneoでは、関連付けられたソフトでファイルを開かせることができます。」

とも　「なるほど、そんなこともできるんだね」

[ファイル名]を開け

ねお　「上のプログラムは、エクスプローラで開いたときと同じソフトが起動してファイルを開くようになっています。」

とも　「ファイルを指定すれば、そのソフトを起動してくれるんだね」

　

ねお　「そう、「開け」命令は他にも、インターネットアドレスやメールアドレスを指定することもできます。次の例のように[ファイル名]の部分をインターネットアドレスやメールアドレスに変えるだけです。」

「http://○○○」を開け
「mailto:○○○」を開け

とも　「ホームページへアクセスするのも「開く」命令なんだね」

ねお　「メールアドレスを指定したときは、新規作成画面を表示してくれます。

ファイルやフォルダの一覧を取得する

TTSneoでは、配列を利用する命令・関数が多く用意されています。

ファイルやフォルダに関係する関数は、次のようなようなものがあります。
これらの関数は、指定したフォルダにあるファイルやフォルダを配列として表します。

ファイル一覧（[フォルダ名]）
フォルダ一覧（[フォルダ名]）
全ファイル一覧（[フォルダ名]）
全フォルダ一覧（[フォルダ名]）

例えば、マイドキュメントにあるフォルダを調べるには、次のように書きます。
このプログラムは、マイドキュメントにある、1番目のフォルダの名前を表します。
２番目以降のファイルは、「配列（２）を表示」など書くことで取得できます。

配列は、フォルダ一覧（マイドキュメント）
配列（１）を表示

また、これらの関数には、ワイルドカードと呼ばれるフィルタをかけることもできます。
つまり、特定の種類のファイルの一覧を調べることができます。

例えば、マイドキュメントにあるテキストファイル(txt)を調べるには、次のように書きます。
このプログラムは、マイドキュメントの中のすべてのフォルダにあるテキストファイルを表します。

配列は、全ファイル一覧（「［マイドキュメント］*.txt」）
配列を表示

ファイル名の部分にある“*”は、文字数にこだわらない任意の文字を表し、txtが拡張子を表します。
拡張子とは、ファイル名の最後の.で区切られた３文字のことです。

拡張子を調べるには・・・

Windowsの既定値では、拡張子が表示されていないことがあります。
拡張子を知りたい場合は、ファイルのプロパティを見るか、
フォルダオプションで拡張子を表示させるように設定してください。

設定するには、エクスプローラを開いて、「ツール」メニューの「フォルダオプション」を開きます。
「表示」タブを選び、詳細設定から「登録されている拡張子は表示しない」のチェックをはずします。
