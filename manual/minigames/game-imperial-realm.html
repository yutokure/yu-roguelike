ミニゲーム: インペリアル・レルム

概要

インペリアル・レルムは、ミニゲームAPI向けに実装された AoE2 風リアルタイムストラテジーです。 村人を統率して資源を確保し、建物を建てて軍備を整え、迫り来る襲撃ウェーブを凌ぎながら敵帝国のタウンセンターを陥落させることが目標です。 帝国士気（Imperial Morale）と時代進化を活用し、攻撃補正を上げつつ防衛と反攻を両立させましょう。

  • カテゴリ: シミュレーション / RTS
  • プレイ時間目安: 8〜15 分（難易度・プレイスタイルによって変動）
  • クリア条件: 敵タウンセンターの破壊
  • 敗北条件: 自軍タウンセンターの破壊、または村人が全滅

ゲームの流れ

  1. ゲーム開始時点で自軍タウンセンターと完成済みの家 2 棟、村人 3 名が配置され、初期資源と周囲の資源ノード（食料・木材・金・石）が与えられます。敵陣には強化されたタウンセンターと兵舎が初期配置されています。
  2. 村人に資源採集を指示しつつ、家や軍事施設を建設して人口上限と部隊生産能力を拡張します。
  3. タウンセンターで時代進化を研究し、新しい建物・部隊・攻撃補正を解禁します。
  4. 定期的に発生する敵ウェーブを迎撃し、敵主力がいないタイミングで反撃して敵タウンセンターを破壊しましょう。最終ウェーブ後は敵司令官が登場します。

難易度によってウェーブ間隔と敵耐久が変化します。序盤の資源配分と建物配置を安定させることが、終盤の反攻に直結します。

操作と UI

  • 左クリック: ユニット・建物を選択。ドラッグで範囲選択。
  • 右クリック: 選択中のユニットに移動 / 採集 / 攻撃命令を発行。対象が敵ユニット・建物なら攻撃、資源ノードなら採集、空き地なら移動命令になります。
  • アクションボタン: 選択した村人や建物に応じた建設・訓練・時代進化コマンドが表示されます。建設コマンドを押した後はキャンバス上で建設位置を左クリックします。
  • Esc キー: 建設モードをキャンセル。

画面上部には資源と人口、時代進行状況、帝国士気、次のウェーブまでの残り時間が表示されます。 左側は作戦ログ、右側はウェーブ情報とミニマップ、下部は選択情報とアクション一覧です。 ミニマップには資源・自軍/敵軍のユニットと建物が色分けで表示され、戦況把握に役立ちます。

資源と経済

  • 資源種別は 食料 / 木材 / 金 / 石 の 4 種。資源表示はローカライズに対応しています。
  • 村人の基礎採集レートは 食料 10、木材 9、金 8、石 7 / 秒換算で、資源ノードが枯渇するとログで通知されます。
  • 資源ノードの初期配置例: 食料 360、食料 280、木材 420 ×3、金 360、石 260。
  • 資源採集 100 ごとに 5 EXP を獲得し、採集担当ユニットはノードが尽きるとアイドル状態に戻ります。
  • 人口上限はタウンセンター基礎 10 + 建設済み家 1 棟あたり +5 で増加します。

時代システム

タウンセンターを選択すると「時代進化」ボタンが表示され、以下の段階で研究できます。研究中は進捗ゲージが HUD に表示されます。

段階 要約 研究コスト 研究時間 攻撃補正
開拓期 初期段階。生存と基盤構築を重視。 なし 即時 +0%
封建期 防衛線と歩兵を強化。 食料 500、金 200 45 秒 +5%
城塞期 重装兵と騎兵で攻勢へ転じる。 食料 800、金 450 60 秒 +10%
帝国期 最先端軍制で優位を確立。 食料 1200、金 750、石 300 75 秒 +18%

研究開始時と完了時に帝国士気が上昇します。研究中にタウンセンターを失うと進化は中断されるため、防衛体制を整えてから実行しましょう。

帝国士気（Imperial Morale）

  • 士気ゲージは 0〜160 の範囲で推移し、攻撃補正は 1 + Σ(時代ボーナス) + 0.04 × 鍛冶場数 + 士気 / 200 で計算されます。
  • 士気は時間経過で減衰し、90 以上では毎秒 0.42、60〜89 では 0.26、59 以下では 0.14 ずつ自然減少します。
  • 建物完成 (+6、鍛冶場は +8)、ユニット完成 (+3)、時代研究開始 (+4)・完了 (+14)、ウェーブ撃退 (+12)、敵撃破 (+1.5) などで上昇します。
  • 味方ユニット喪失 (-6)、自軍建物破壊 (-12) などで低下します。士気が高いほど攻撃力が伸びるため、被害を抑えながら攻勢を維持しましょう。

敵ウェーブと司令官

一定時間ごとに敵ウェーブがスポーンし、以下の構成で攻めてきます。すべて撃退すると補給物資として全資源 +40 と多めの EXP を獲得します。

ウェーブ 構成
第1波 略奪兵 ×6、騎馬弓兵 ×2
第2波 略奪兵 ×8、騎馬弓兵 ×2、破城槌 ×1
第3波 略奪兵 ×10、騎馬弓兵 ×3、破城槌 ×2、騎士 ×2
第4波 (HARD) 略奪兵 ×12、騎馬弓兵 ×5、破城槌 ×2、騎士 ×3

すべての規定ウェーブが終了すると、敵司令官（HP 280、攻撃 18）が戦場に現れます。司令官を含む敵勢力を殲滅し、敵タウンセンターを破壊すると勝利です。 防衛に失敗して自軍タウンセンターが破壊されるか、村人が全滅すると即敗北になります。

報酬と EXP

  • 建物完成で 12 EXP、ユニット訓練開始で 4 EXP（難易度補正適用）、敵撃破でユニットごとの EXP を獲得します。
  • 資源採集 100 ごとに 5 EXP、ウェーブ撃退ボーナスは 120 / 180 / 260 / 320 EXP（難易度補正あり）。
  • 難易度係数によって最終獲得 EXP が増減するため、慣れてきたら HARD に挑戦すると効率良く経験値を稼げます。

攻略のヒント

  • 序盤は村人 3 名をそれぞれ食料・木材・木材に割り当て、早めに家と兵舎を建てると人口不足と初動防衛を両立できます。
  • 封建期までに塔か追加兵舎を準備し、破城槌対策として槍兵や騎士をタイミングよく投入しましょう。
  • 鍛冶場は士気と攻撃補正を同時に伸ばせるため、城塞期に入ったら最優先で建設すると反攻が楽になります。
  • 敵タウンセンターは HP 2200 と堅牢です。ウェーブを捌いた直後に主力を温存しつつ攻め込み、破城槌や騎士で集中攻撃するのが効果的です。

トラブルシューティング

  • 建設モード中に資源が不足している場合、配置クリック時にログで警告され建設は行われません。必要資源を確認してから再試行してください。
  • 建物やユニットが選択できない場合は、自軍のものかどうかを確認してください（敵ユニットは命令不可）。
  • 研究中のタウンセンターが破壊されると進化がキャンセルされます。防衛が難しい場合は塔や城塞期ユニットで周辺を固めましょう。
