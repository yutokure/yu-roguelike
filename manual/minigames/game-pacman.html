ミニゲーム: パックマン風

概要

迷路内に散らばったペレットをすべて集めつつ、追跡してくるゴーストを回避するクラシックなドットイート・アクションです。 固定レイアウトのステージを縦横無尽に駆け抜け、ペレットをコンプリートして高スコアを目指します。

準備とゲーム開始

  1. タイトルメニューから パックマン風 を選択します。
  2. 必要に応じてホスト設定で難易度（EASY / NORMAL / HARD）を切り替えます。難易度によって残機やゴーストの移動速度が変化します。
  3. ゲームを開始するとプレイヤーキャラクターが中央左寄りの位置に配置され、ゴーストがゴーストハウスから動き出します。

プレイ中はホストの R ショートカットが無効化されます。ゲーム終了後に自動的に再び有効になります。

操作方法

  • ← / → / ↑ / ↓：進行方向を指定します。進行方向は次に到達する交差点で反映されます。
  • ゲーム終了後の R：難易度設定を保ったままステージを即座にリスタートします。

マウス操作は不要で、キーボードのみで完結します。

ルールとスコア

  • 小さなペレットを 1 つ取るたびに +0.5 XP を獲得し、カウントがスコアに反映されます。
  • ステージ上の全ペレットを取り切ると追加で +100 XP を獲得してクリアになります。
  • 画面左上の HUD では残機と取得済みペレット数 / 残りペレット数が常時表示され、進行状況を把握できます。
  • 迷路中央のトンネルを通ると左右端がループして反対側にワープできます。移動経路をショートカットする際に活用しましょう。

ライフとゲーム終了条件

ゴーストに接触すると残機を 1 つ失い、残機が残っている場合はプレイヤーとゴーストの初期配置が復帰してプレイが継続されます。 残機が 0 になるとゲームオーバーとなり、画面中央に結果オーバーレイが表示されます。クリア時も同様にゲームが停止します。

難易度による違い

  • EASY：残機 3。ゴーストの移動速度が最も遅く、初心者向けです。
  • NORMAL：残機 3。標準的な速度で、基本の遊び方を学ぶのに適しています。
  • HARD：残機 2。ゴーストの移動速度がさらに上昇し、追跡から逃げ切るには綿密なルート取りが求められます。

攻略のヒント

  • 進行方向の入力は次に到達する交差点で判定されるため、早めに方向キーを押してスムーズに曲がりましょう。
  • ゴーストは一定の確率で進路を再評価し、プレイヤーに最も近づける方向を選びます。追跡されていると感じたら遠回りして距離を取りましょう。
  • 中央トンネルを使うとゴーストとの距離を一気に引き離せます。追われた時の退避ルートとして覚えておくと安心です。
