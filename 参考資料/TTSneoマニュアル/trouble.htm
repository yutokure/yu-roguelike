TTSneoデザイナが起動しない

必要なファイルが不足している場合や、管理者権限でないユーザでログインした場合に、
TTSneoが起動しないことがあります。

「Visual Basic 6.0ランタイムが必要です。ダウンロードしますか？」

TTSneoデザイナの起動に必要なファイルが不足しているため、
Visual Basic 6.0ランタイムをインストールする必要があります。

[はい]をクリックすると、TTSneo公式サイトにあるVisual Basic 6.0ランタイムを自動的にダウンロードし、インストールを行います。

　

ランタイムをインストールしたにもかかわらず
「Visual Basic 6.0ランタイムが必要です。・・・」と何度も表示される

Windows VistaやWindows XPの制限ユーザ(Users権限)では、TTSneoデザイナが起動できないことがあります。

このような場合は、次の手順で必要なプログラムの再登録を行ってください。

１．管理者権限でログインします。

２．スタートメニューの「ファイル名を指定して実行」（検索バー）で、
「regsvr32 /u MSFLXGRD.OCX」と入力して、[OK]をクリックします。

　　「成功しました」または「Successful」などと表示されたことを確認します。

３．スタートメニューの「ファイル名を指定して実行」（検索バー）で、
「regsvr32 MSFLXGRD.OCX」と入力して、[OK]をクリックします。

　　「成功しました」または「Successful」などと表示されたことを確認します。

　

※また、ユーザアカウントの権限が変更可能であれば、PowerUser（標準ユーザ）または管理者に設定することでも解消します

※Windows95/98/Meや管理者権限で、メッセージが何度も表示される場合は、再度Visual Basic 6.0ランタイムをインストールしてください

　

「〜が見つかりません。
コンポーネントXXX.OCXまたはその依存関係のひとつが適切に登録されていません。」

必要なファイルが不足しているか正常に登録されていないため、
Visual Basic 6.0ランタイムをインストールする必要があります。
Visual Basic 6.0ランタイムをインストールしてください。

　

「TTSneoDLLが利用できません。」
「ActiveXコンポーネントが作成できません。」
「ActiveXコンポーネントはオブジェクトを作成できません。」
「オートメーションエラーです。」

TTScript.dllが正しくインストールされていません。
TTSneoを再度インストールしてください。

　

「メモリが不足しています」

Windowsが問題なく動作すればTTSneoは動作します。
メモリの容量が十分にある場合、ファイルが壊れているか、正しくダウンロードされていない可能性があります。
再度ダウンロードしてインストールしてください。

　

最新版をお試しください

TTSneo公式サイトから、最新版を入手することができます。

TTSneoは日々改良していますので、雑誌などで入手された場合は最新号でも編集の関係で、古いバージョンである場合があります。

Webサイトから最新版を入手して、いずれの方法を試してみても問題が解決されない場合は、掲示板でご報告をお願いします。

  • 症状は、できるだけ詳しくお願いします
  • エラーが表示される場合はエラーメッセージを正確にお願いします。
    （「エラーが出る」だけでは、分かりませんので具体的操作やメッセージの内容）
    （強制終了の場合の[詳細]表示(例外発生アドレス)は不要です）
